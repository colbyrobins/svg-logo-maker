## Installation

```Bash
git clone git@github.com:colbyrobins/svg-logo-maker.git

cd svg-log-maker/Devlop

npm install

```

## Usage

```Bash
// execute npm start to start the application.

$ npm start

> start
> node index.js

// Answer the prompts
? Please enter your text DXD
? Please your text color white
```
